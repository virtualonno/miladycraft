### ✦ Mod Additions

  * [Colossal Chests](https://www.curseforge.com/minecraft/mc-mods/colossal-chests): BIG chests
  * [Cyclops Core](https://www.curseforge.com/minecraft/mc-mods/cyclops-core): library mod for Colossal Chests, amongst others
  * [Functional Storage](https://www.curseforge.com/minecraft/mc-mods/functional-storage): Storage drawers-type storage
  * [Titanium](https://www.curseforge.com/minecraft/mc-mods/titanium): library mod for Functional Storage, amongst others
  * [CrashExploitFixer](https://legacy.curseforge.com/minecraft/mc-mods/crashexploitfixer): self-explanatory, fixes a certain crash exploit present before MC 1.21.1
  * [Packing Tape](https://www.curseforge.com/minecraft/mc-mods/packing-tape): pack up block entities so that they can be picked up and moved around with their contents
  * [EventJS](https://www.curseforge.com/minecraft/mc-mods/eventjs): Natively reloadable ForgeEvents for KubeJS like is implemented in MC 1.21

### ✦ Mod Removals

  * [Casinocraft](https://www.curseforge.com/minecraft/mc-mods/casinocraft): preliminary removal for Miladycraft 1.6
  * [Suggestion Tweaker](https://www.curseforge.com/minecraft/mc-mods/suggestion-tweaker): possibility for server crash on startup

### ✦ Features & Changes

  * A Miladycraft-custom death message system! Currently implemented as proof-of-concept through a variety of new gun-related death messages, inspired by arena shooters; let me know if you have any issues
  * Disabled gamerule showDeathMessages to account for custom death message system
  * Bamboo recipe to turn Quark's bamboo blocks back into regular bamboo
  * Temporary set direct server IP in multiplayer list until A record is added back to miladycraft web domain
  * Bluemap out of order for the time being
